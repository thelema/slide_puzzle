## Project Overview

This is an sliding tile game, where the player moves tiles out of the board by sliding them out of same-colored facets on the edge of the board.  The codebase, a mixture of python and  is extremely new and is still approaching its first functional version.

## Coding rules

Don't access files outside the workspace directory.
Ask the user to do anything requiring sudo permissions.
All python functions should have type annotations and attrs/dataclass-based classes should be used by default.
